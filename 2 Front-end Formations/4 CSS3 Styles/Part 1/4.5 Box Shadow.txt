4.5 Box Shadow

Add a subtle box-shadow to our .post. We want it to be a drop shadow that is 0 to the right, 1px down, has a 1px blur-radius, and a color of #000.


style.css

.post {

}

Answer:

.post {
  box-shadow: 0px 1px 1px #000;  
}
