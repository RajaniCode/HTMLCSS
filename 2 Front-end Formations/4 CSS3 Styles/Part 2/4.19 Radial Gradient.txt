4.19 Radial Gradient

http://caniuse.com/#feat=css-gradients

Let’s add a radial-gradient to our .masthead. Write a radial-gradient with two color-stops, #1e3e4d and #102028. We will just use the default ellipse as the gradient shape.

style.css

.masthead {

}

Answer:

.masthead {
  background: radial-gradient(#1e3e4d, #102028);
}
