4.17 Linear Gradient

http://caniuse.com/#feat=css-gradients

We want to add a subtle linear-gradient to our .nav. Create a linear-gradient with two color-stops, #3e606f and #324d59. For now, we’ll leave the gradient going in the default direction (to bottom or 180deg).

style.css

.nav {

}

Answer:

.nav {
  background: linear-gradient(#3e606f, #324d59);
}
