Convert to Percentage


From the CSS styles given, convert .content to a percentage. Give your answer to 6 decimals. Use this Flexible Math tool to help: http://responsv.com/flexible-math/

HTML SOURCE

<div class='container'>
  <div class='content'>
  </div>
</div>

CSS SOURCE

.container {
  width: 960px;
}
.content {
  width: 550px;
}

STYLE.CSS

.content {
  width: 550px;
}

Answer:

.content {
  width: 57.291666%; /* 550 / 960 * 100 */
}
