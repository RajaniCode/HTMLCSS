1.3 Why Bootstrap?

Why would someone want to use Bootstrap?

    Bootstrap eliminates the need to write lengthy HTML code, curtailing load times.

    Bootstrap provides pre-built styling and functionality, making site-building faster and easier.

    Bootstrap eliminates the need to be aware of older browser limitations when building websites.

    Bootstrap is a framework that will automatically build the entire website when imported into HTML code.

Answer: Bootstrap provides pre-built styling and functionality, making site-building faster and easier.
