5. Lastly, find the .btn .content selector which will target all content elements within a button. Make these transition their top property over whatever duration you want.

Answer:

  transition: all 0.4s;
}

.top.content {
 top: 0;

 top: -50px;
}

	top: -50px;
}

	top: 0;

.btn .content {
	transition: top 2s;
